Meat Inventory Dashboard - PHP + MySQL (XAMPP)

How to run locally:
1. Install XAMPP (https://www.apachefriends.org/)
2. Start Apache and MySQL from XAMPP control panel.
3. Copy the folder 'meat_inventory' into XAMPP's htdocs directory (e.g., C:\xampp\htdocs\).
   Or run directly from this folder if you configure Apache's DocumentRoot to it.
4. Open phpMyAdmin: http://localhost/phpmyadmin
5. Import the SQL file 'meat_inventory.sql' (or run its SQL) to create the database and tables.
6. Open in browser: http://localhost/meat_inventory/index.php

Default DB connection uses user 'root' with empty password. If your MySQL has a password, update db.php accordingly.
